# ActiveMQ_Crack
## 概述
ActiveMQ系列漏洞探测利用工具，包括ActiveMQ 默认口令漏洞及ActiveMQ任意文件写入漏洞（CVE-2016-3088），支持批量探测利用。
## 使用方法
批量探测：文件中逐行写入需要检测的URL，参数-TF 指定文件 -t 并发数量

单个目标探测：-T http://x.x.x.x
